# ☕ Vue Coffee Store

Projeto de uma **loja fictícia de café** desenvolvido com **Vue.js**, com o objetivo de praticar conceitos de front-end e organização de aplicações web.

## 🧾 Sobre o projeto
A Vue Coffee Store simula uma cafeteria online, permitindo a visualização de produtos, informações sobre cafés e uma interface moderna e responsiva.  
Este projeto foi desenvolvido com fins **educacionais e de portfólio**.

## 🚀 Funcionalidades
- Listagem de cafés/diversos
- Página inicial com apresentação da loja
- Componentes reutilizáveis
- Navegação entre páginas

## 🛠️ Tecnologias utilizadas
- **Vue.js**
- **JavaScript**
- **HTML5**
- **CSS3**

## ⚙️ Como executar o projeto
```bash
# Clone o repositório
git clone https://github.com/duda-mm/Coffee-Shop-Vue.git

# Acesse a pasta do projeto
cd Coffee-Shop-Vue

# Instale as dependências
npm install

# Execute o projeto
npm run serve
