# ☕ Vue Coffee Store

Projeto de uma **loja fictícia de café** desenvolvido com **Vue.js**, com o objetivo de praticar conceitos de front-end e organização de aplicações web.

🔗 **Acesse o projeto online:** ([Coffee Shop](https://coffee-shop-vue-seven.vercel.app/))

## 🧾 Sobre o projeto
A Vue Coffee Store simula uma cafeteria online, permitindo a visualização de produtos, informações sobre cafés e uma interface moderna e responsiva.  
Este projeto foi desenvolvido com fins **educacionais e de portfólio**.

## 🚀 Funcionalidades
- Listagem de cafés/diversos
- Página inicial com apresentação da loja
- Componentes reutilizáveis
- Navegação entre páginas

## 🛠️ Tecnologias utilizadas
- **Vue.js**
- **JavaScript**
- **HTML5**
- **CSS3**

## 📸 Modelagem 

[Projeto modelo no Figma](https://www.figma.com/design/1eivyVR3ZGI8b8frH3IRSt/Coffee-Shop-Vue?node-id=0-1&t=E7FRdJU11tU2VPkU-1)

## 🎯 Objetivo

Este projeto tem como objetivo reforçar conhecimentos em:

- Componentização no Vue

- Organização de código

- Desenvolvimento de interfaces web

## 📌 Observações

Projeto fictício, sem fins comerciais.


## ⚙️ Como executar o projeto
```bash
# Clone o repositório
git clone https://github.com/duda-mm/Coffee-Shop-Vue.git

# Acesse a pasta do projeto
cd Coffee-Shop-Vue

# Instale as dependências
npm install

# Execute o projeto
npm run serve
